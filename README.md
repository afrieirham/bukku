# [Bukku.my](https://bukku.my) Assessment

## Running the app
1. Clone the repo
2. Run `bun install`
3. Run `bun run dev`

## Technical Spec
- Built with [T3 stack](https://create.t3.gg)
  - Next.js
  - Tailwind
  - tRPC
  - TypeScript
  - Prisma
- SQLite
- [ShadCN UI](https://ui.shadcn.com) library
- Vitest testing library
